## 视频1：新手如何报名接单

**功能模块**：注册 + 接单流程 + 签到签退
**教程类型**：新手教程
**统一 Slogan**：找兼职，上【蜂职速聘】

### 文案/脚本

```
[开场]
蜂职小课堂：新手如何报名接单？

[步骤1 — 注册完善资料]
首先，需要先注册个人信息。
打开【蜂职速聘】微信小程序，选择【身份】。
进入页面可以看到很多服务订单。
如果想接单，找到【我的】界面，完善资料。
第一次接单需要完善资料（现可以从主页面完善）：
  - 上传近期正面照片
  - 身份证正反面
  - 健康证（选填）
  - 填写个人简介
确认无误后点击下一步。

[步骤2 — 填写意向]
填写【意向岗位】：后厨帮工、服务员、撤台、洗碗工等
填写接单时段
确认无误后再次点击"下一步"

[步骤3 — 接单位置]
填写【接单位置】，设置【接单范围】
提交审核通过后即可接订单
提交后会弹出兼职群二维码，可加入兼职群
在接单过程中出现异常情况，客服小助手可在群里帮忙对接

[步骤4 — 抢单]
完善资料后就可以接单啦！
在接单首页"抢单池"选择适合的订单
点击【立即抢单】，查看【订单须知】
点击【开始接单】即可锁定，订单状态为【待确认】
请仔细阅读相关的平台规则

[步骤5 — 订单确认]
若商家拒绝：平台会电话通知，可返回抢单池重新选择
若商家未确认：系统将在服务开始前2小时自动确认
并通过语音电话提醒订单进入【待服务】界面
此时可以在平台上选择告诉商家【我已出发】
（如果没有点击已出发，后续无法选择签到）

[步骤6 — 签到签退]
到工作地点后，马上在订单里点【签到】
干完活儿别忘了立刻点【签退】确认完工时间
特别提醒：
  - 迟到或早退，商家有权申请部分扣款
  - 一旦产生纠纷需平台协调，工资到账时间会推迟
  - 不签到/签退，系统无法核算工作时长，无法结算工资
  - 不签到/签退会被标记为一次"爽约记录"

[步骤7 — 星级体系]
平台根据三个维度划分星级（30天为周期）：
  接单量 + 信用分 + 服务质量

  | 星级 | 接单条件 | 时薪 |
  |------|----------|------|
  | 3星  | 新用户 / 接单＜8 | 基础 |
  | 4星  | 8≤接单≤15 | 更高 |
  | 5星  | 接单≥16 | 最高 |

  星级越高，可接订单的时薪也越高

[结束]
找兼职，上【蜂职速聘】。
```

### 关键操作点
- 完善资料 → 意向岗位 → 接单范围 → 提交审核
- 抢单 → 待确认 → 已出发 → 签到 → 签退
- 资料不完整无法接单；不签到签退 = 爽约

### 注意事项
- 接单后无故取消/爽约将被记录，严重者永久封号
- 爽约后，下一单需交付保证金才能接单
- 看清楚订单信息再确认接单，不可盲目接单
- 不符合预期的订单（时间冲突、内容不合适、交通不便、单价不满意）不要强行接单

## 视频2：如何在平台上多赚钱

**功能模块**：星级晋升 + 抢单技巧
**教程类型**：进阶教程

### 文案/脚本

```
[开场]
一分钟教会你【蜂职速聘】平台上怎么多赚钱？
两个诀窍，别错过重点：

[诀窍1 — 星级晋升]
为了让"蜂友"们在平台有更好的成长空间、获得更高的报酬，
平台根据接单量、信用分、服务质量三个维度划分星级：

  | 星级 | 接单条件 | 时薪 |
  |------|----------|------|
  | 3星  | 新用户 / 接单＜8 | 基础 |
  | 4星  | 8≤接单≤15 | 更高 |
  | 5星  | 接单≥16 | 最高 |

晋升规则（30天周期）：
  3星→4星：30天内接单≥8单
  3/4星→5星：30天内接单≥16单
  5星封顶

特别福利：商家一单5星好评，可抵2单数量！

[诀窍2 — 抢单技巧]
什么时间订单最多？
  早上8点到下午2点，傍晚5点到晚上10点

怎么挑选合适的订单？
  根据等级要求、岗位距离、单价、订单详情内容挑选

怎么优先抢单？
  保持高信用分可以优先抢单
  接单后不迟到、不早退、不爽约
  认真干活，争取好评加分
  信用分越高越优先抢单

平台福利：不定时上线各种福利活动（如平台补贴等）
及时关注参与活动，可以赚更多钱

[结束]
找兼职，上【蜂职速聘】。
```

### 关键操作点
- 30天接单≥16单 → 5星最高时薪
- 5星好评抵2单
- 高信用分 = 优先抢单 = 更多收入

## 视频3：信用分怎么用

**功能模块**：信用分体系
**教程类型**：规则解读

### 文案/脚本

```
[开场 — 用户痛点]
有"蜂友"们问，在招工页面明明看到很多服务订单，
但是自己却接不了了，平台提示是信用分不够，不能接单。
这个信用分究竟是什么？要怎么才能获取信用分呢？
那么这条视频你可要仔细看！

[什么是信用分]
信用分是"蜂友"们在【蜂职速聘】平台上个人信用高低的直观体现。
信用分越高 → 履约行为越好
信用分越低 → 履约行为越差

[初始分]
新用户注册：初始信用分 = 100分

[加分项]
  - 正常接单、完单
  - 完成多天单
  - 收到商户好评

[扣分项]
  - 多次只报名不接单
  - 服务前取消订单
  - 爽约
  - 迟到早退

[信用分的作用]
分值越高、等级越高 → 优先接单，服务单价越高
分值越低 → 只能捡别人挑剩下的单

[总结]
接单服务好、态度好、信誉好 → 优先接单，接优质单
差行为 → 限制接单，或接普通单

[红线规则 — 特别注意]
平台严禁以下行为（触及即无限期封禁）：
  1. 溜单
  2. 虚假打卡
  3. 辱骂、威胁、骚扰老板

蜂友们一定要牢记平台规则，
违反信用分、触及平台红线的事儿咱千万不能做！

[结束]
找兼职，上【蜂职速聘】。
```

### 关键操作点
- 初始100分，好行为加分，差行为扣分
- 三条红线：溜单、虚假打卡、辱骂威胁骚扰 → 永久封禁

## 视频4：如何结算当日报酬并提现

**功能模块**：结算 + 提现
**教程类型**：操作指南

### 文案/脚本

```
[开场 — 直击痛点]
订单服务结束后报酬如何结算？结算后如何提现？
"蜂友"们最关心的问题，本条视频一次性给大家说清楚，
一定要认真看完哦！

[结算报酬]
当天服务完成打卡后：
  场景1：老板在现场
    → 尽量与老板沟通，现场确认报酬并结算

  场景2：老板不在现场
    → 服务过程中无重大争议
    → 平台在2小时后自动结算报酬

[报酬提现]
在【蜂职速聘】里：
  点【我的】 → 点击【提现】 → 确认提现金额
  → 微信账户进行提现
  → 正常情况下提现金额即刻到账

[结束]
找兼职，上【蜂职速聘】。
```

### 关键操作点
- 老板在场：当场确认结算
- 老板不在：2小时后自动结算
- 提现路径：我的 → 提现 → 微信到账

## 视频5：哪些行为会被扣分和降等级

**功能模块**：违规处罚
**教程类型**：规则解读

### 文案/脚本

```
[开场]
"蜂友"们一定要关注的问题：哪些行为会被扣分和降等级？

[一、降等级规则]

  │ 当前等级 │ 保持条件（30天）  │ 不达标的后果  │
  │----------│-------------------│--------------│
  │ 4星      │ 接单≥6单          │ 降至3星      │
  │ 5星      │ 接单≥12单         │ 降至4星      │

  **触发立即降级的两种情况：**
  - 被商家投诉1单 → 降星一级，星级重置
  - 爽约1次 → 降星一级，星级重置

  降级规则说明：
  - 降级仅限一级（5星→4星，4星→3星）
  - 被降级后，需按晋升规则重新晋级
  - 3星爽约1次 → 需交保证金10元，正常接1单完成后可退回

[二、扣分行为]

  - 服务前取消订单
  - 接单后迟到、早退、爽约、缺卡、旷工
  - 兼职自己不取消订单，联系老板取消订单

[三、红线规则 — 触及永久封禁]

  1. 和老板交换联系方式
  2. 辱骂、骚扰、威胁老板或客服
  3. 虚假打卡、未实际工作、或工作期间做其他事情（如玩手机等）

  触及以上三条红线，平台将无限期封禁相关账号！

[结束]
找兼职，上【蜂职速聘】。
```

### 关键操作点
- 4星保级：30天≥6单；5星保级：30天≥12单
- 投诉/爽约 = 直接降级
- 三条红线永久封禁
